# Spot-Finder

Discover and save the perfect spots.

## Overview

Spot-Finder helps you find, rate, and remember great locations — whether for surfing, fishing, hiking, parking, photography, or anything else.

## Features (Planned)

- Search and map spots
- User reviews and ratings
- Save favorites
- Filters by category, distance, conditions
- Share spots with friends
